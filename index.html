La especificación ECMAScript

Normas y reglas de funcionamiento de Javascript

Ver Referencia

La especificación ECMAScript es un documento que define el estándar para el lenguaje de programación JavaScript. Es mantenida por ECMA International, una organización de estándares de tecnología.

ECMAScript es la base sobre la cual se construye JavaScript, y define las reglas y características que deben cumplirse para que un lenguaje sea considerado compatible con esta especificación.

Cada versión de ECMAScript introduce nuevas características y mejoras al lenguaje, manteniendo la compatibilidad con versiones anteriores.

Los tipos de datos son la naturaleza del contenido de una variable o constante.

En la programación debemos tener varios conceptos claros porque estaremos utilizándolos todo el tiempo y es importante no confundirlos:

  1. nombre de una variable
  2. contenido de una variable
  3. tipo de dato de la variable
